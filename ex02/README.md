# 第2回
## 電卓（ex02/calc.py）

## 重要
### **Matplotlib, PLTが入っていない場合にはconda installなどを用いてモジュールをいれてから実行してください。**
### 電卓機能
- 四則演算が可能です。
- 四則演算記号を2連続で入力することができません。
- ACでリセットできます
- 一番右のボタンを用いることでグラフの描画を行えます
※グラフ描画における注意：現在(2022/11/29)はy=2xのような切片のない正の一次関数しか表示できません。また、グラフの画像が同一フォルダ内に保存されます。
